Tunnelbana Simulation

Python program to simulate trains running back and forth on different lines, similar to a tunnelbana system. The simulation uses a discrete, turn-based model where stations are one unit of time apart. In each stage (unit of time), a train either moves to the next station or stays at its current station (if delayed). The map of train stations is represented as a graph, where nodes are stations, and edges represent the tracks connecting them.

Technologies: Python

Developed at: Stockholm University
